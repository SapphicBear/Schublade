# Schublade

Schublade is a file sharing and downloading application that allows users to log in, upload files and share with others!

This project was made for the [Odin Project's Node.js Backend Course](https://www.theodinproject.com/paths/full-stack-javascript/courses/nodejs).

## Technologies

- Prisma ORM
- EJS
- Express.js
- PostgreSQL
- Node.js
